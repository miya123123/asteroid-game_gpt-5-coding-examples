# NEON ASTRO 
OpenAIの公式GPT-5コーディング例（asteroid-game）を使用して開発した小惑星ゲームとなります。

# デモ
[Demo | NEON ASTRO](https://miya123123.github.io/neon-astro/)

# 使用技術
### LLM
主にgpt-5（Reasoning Effort:High）をコーディングの生成に使用しました。
### 開発ツール
OpenAI Codex CLIを使用しました。
